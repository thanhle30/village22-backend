# Backend for Village Start-up Project
## Description
- Built the backend for Village web product
  + Village is an educational website service that matches a qualified college student as a teaching assistant to a middle school teacher in need of support (for more details, see [these slides](/VillageOverview.pdf)).
  + Developed as a course project at Fairfield University in 2021-2022.
 
## Tech Stack
- ASP.NET Core
- C#
- SQL Database

## Results
- Completed fully functioning backend application: [Demo Video](Demo-FinalVersion.mp4)
- Village won Honorable Mention at [Fairfield Startup Competition 2022](https://www.fairfield.edu/startup/2022.html).
